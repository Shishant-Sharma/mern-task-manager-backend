# MERN Task Manager

A full-stack task management application built with:

- MongoDB Atlas
- Express.js
- React.js
- Node.js
- JWT Authentication
- bcrypt Password Hashing

## Features

- User Registration
- User Login
- Create Tasks
- Update Tasks
- Delete Tasks
- Protected Routes

## Tech Stack

Frontend: React
Backend: Node.js, Express
Database: MongoDB Atlas
Authentication: JWT
